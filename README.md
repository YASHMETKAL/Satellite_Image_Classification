# Satellite Image Classification (Flask + CNN)

Simple project where user uploads a satellite image and gets prediction.

Classes:
- Agriculture
- Forest
- Urban
- Water

## Project Files

- app.py
- templates/index.html
- templates/result.html
- static/style.css
- static/uploads/
- model/model.h5

## Run

1. install packages

```bash
pip3 install -r requirements.txt
```

2. start flask app

```bash
python3 app.py
```

3. open browser

```text
http://127.0.0.1:5000
```
